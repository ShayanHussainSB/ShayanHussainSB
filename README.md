### bunWay — Bun-Native HTTP Router
<a href="https://github.com/bunwaylabs/bunway"><img src="https://img.shields.io/badge/GitHub-bunwaylabs/bunway-181717?logo=github&style=flat-square"></a>

- Middleware support, path parameters, per-route body parsing
- Built for performance-critical Bun applications

### GraphQL Multi-Language SDK Generator

- Built AST-based interpreter generating **type-safe client libraries in 4 languages** (Rust, TypeScript, C#, C++)
- Automated publishing pipeline to **npm, crates.io, and NuGet** via GitHub Actions
- Reduced SDK maintenance overhead across multiple codebases

### Honeycomb Protocol — Solana Gaming Infrastructure

- **Compressed Accounts Architecture** — Combined Bubblegum with custom state compression, achieving **1000x storage cost reduction**
- **Mining Badgers** — On-chain gamification system with token minting, item crafting, and dynamic NFT generation
- Enhanced **4 production programs** (Missions, Staking, Character Manager, Hive Control)

### OKDverse — NFT Collection Launch
