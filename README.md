### Honeycomb Protocol — Solana Gaming Infrastructure

- Built custom minting UI and backend integration

### bunWay — Bun-Native HTTP Router
<a href="https://github.com/bunwaylabs/bunway"><img src="https://img.shields.io/badge/GitHub-bunwaylabs/bunway-181717?logo=github&style=flat-square"></a>
